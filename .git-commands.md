# Git 仓库关联命令

请按顺序执行以下命令来初始化 Git 仓库并关联远程仓库：

## 1. 初始化 Git 仓库（如果还没有初始化）

```bash
git init
```

## 2. 添加远程仓库

```bash
git remote add origin https://github.com/chenbin2015/simpleWebsite.git
```

## 3. 检查远程仓库是否添加成功

```bash
git remote -v
```

## 4. 添加所有文件到暂存区

```bash
git add .
```

## 5. 提交文件

```bash
git commit -m "Initial commit: Vue3 frontend + Spring Boot backend demo"
```

## 6. 推送到远程仓库

```bash
git branch -M main
git push -u origin main
```

## 注意事项

- 如果远程仓库已有内容，可能需要先拉取：
  ```bash
  git pull origin main --allow-unrelated-histories
  ```

- 如果遇到认证问题，可能需要配置 Git 凭据或使用 SSH 方式
